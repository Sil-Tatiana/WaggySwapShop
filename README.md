# WaggySwapShop

WaggySwapShop is an e-commerce application for buying and selling pet products.

- [Features](#features)
- [Technologies](#technologies)
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)

## Introduction

WaggySwapShop is a platform that connects pet owners and pet product sellers. It provides a convenient way for users to browse and purchase pet-related items, as well as for sellers to list and manage their products.

## Features

- User authentication and authorization
- Product listing and search functionality
- Shopping cart and checkout process
- Order management
- User reviews and ratings
- Seller dashboard for managing products and orders

## Technologies

- Node.js
- Express.js
- PostgresSQL
- JWT for authentication
- Stripe for payment processing

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/WaggySwapShop.git
   ```

2. Install the dependencies:

   ```bash
   cd WaggySwapShop
   npm install
   ```

3. Set up the environment variables:

   - Create a `.env` file in the root directory
   - Add the following variables:

     ```plaintext
     PORT=3000
     ```

4. Start the application:

   ```bash
   npm start
   ```

## Usage

- Visit `http://localhost:3000` in your browser to access the application.
- Create an account or log in to an existing account.
- Browse and search for pet products.
- Add products to your cart and proceed to checkout.
- Manage your orders and leave reviews for products.
